# Global Financial Market Dashboard

This project is an interactive financial dashboard built using Python and Streamlit. It allows users to track and compare global market assets including Gold, USD/PKR, Bitcoin (BTC), Ethereum (ETH), and U.S. Stocks.

The dashboard includes real-time data updates, market comparison charts, currency conversion, and automatic short trend summaries to help users quickly understand market direction.

# Features

Live market tracking (Gold, Crypto, Currency, U.S. Stocks)

Compare multiple assets over different time ranges

Currency converter with instant results

Line charts and heatmap visualization

AI-generated short market insights

Clean professional UI design

# Tech Used

Python

Streamlit

Pandas

NumPy

# How to Run

Install required libraries:

pip install -r requirements.txt


Start the app:

streamlit run app.py in cmd


Or double-click:
run_dashboard.bat (Windows)

# Folder Contents

app.py → Main dashboard UI

utils.py → Functions for data fetch & analysis

requirements.txt → Required packages

run_dashboard.bat → One-click launcher
