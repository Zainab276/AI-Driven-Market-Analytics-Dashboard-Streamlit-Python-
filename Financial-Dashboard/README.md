# Sparking-Asia-Financial-Dashboard-Upgraded

Upgraded Sparkling Asia Financial Dashboard - Streamlit app with improved UI, cards, charts, top movers, and PDF export.

## Quick Start (Windows)
1. Unzip folder.
2. (Optional) create virtualenv: `python -m venv .venv` and activate it.
3. Install dependencies: `pip install -r requirements.txt`
4. Double-click `run_dashboard.bat` to launch the dashboard (or run `streamlit run app.py`).
